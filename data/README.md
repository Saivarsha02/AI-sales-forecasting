## Data Folder
This folder contains raw and processed datasets used for sales forecasting.
Raw data is stored as downloaded, while processed data is cleaned and prepared
for modelling.
## Dataset Source
This project uses the Walmart Weekly Sales dataset from Kaggle.
Source:
https://www.kaggle.com/datasets/mikhail1681/walmart-sales
Due to size considerations, raw data files are not stored directly in the
repository. Instructions to download the dataset are provided below.
